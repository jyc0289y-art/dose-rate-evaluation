3. 평가 방법

3.1 기본 가정

비차폐 점선원(bare point source) 가정
선원 인증서가 미확보되어 밀봉캡슐의 재질·두께·형상을 특정할 수 없으므로, 밀봉캡슐에 의한 자기차폐를 전혀 고려하지 않은 최보수적 조건으로 평가한다.
참고: Fe-55 밀봉선원은 5.9 keV Mn Kα X선을 분석용으로 방출하기 위해 Be창 등 저Z 창재를 사용하므로, 캡슐 차폐를 가정하는 것은 부적절하다. 따라서 3핵종 모두 비차폐 선량률 평가로 갈음한다.

3.3 단위 환산 방법

Smith & Stabin [R1]의 선량률정수는 Exposure Rate Constant (R·cm²/(mCi·h)) 단위로 수록되어 있다. 본 평가에서는 다음 환산 과정을 거쳐 μSv·m²/(MBq·h)로 변환하였다.
환산 체인:

Γ [μSv·m²/(MBq·h)] = ΓR [R·cm²/(mCi·h)] × 0.023703

유도:
  • R → μSv: 1 R = 8.77 mGy = 8,770 μGy ≈ 8,770 μSv (Q=1, 광자)
  • cm² → m²: ÷ 10,000
  • mCi → MBq: ÷ 37 (1 mCi = 37 MBq)

  = 8,770 / 10,000 / 37 = 0.023703
보수성 참고: 이 환산은 공기커마(Ka) 기준이며, 실제 인체 유효선량(effective dose)이나 주위선량당량 H*(10)은 에너지에 따라 이보다 낮거나 비슷한 수준이다. 따라서 공기커마 기준 평가는 보수적이다.

3.4 평가 거리

저장구역 경계면(저장함 표면 ~ 최소 0.3 m) 및 1 m(기준 거리)에서 평가한다.

6.4 핵종별 복합 스펙트럼 반가층 산정

§6.2~6.3의 단색 HVL은 개별 방출선 에너지에 대한 것이다. 실제 핵종은 다수의 에너지를 동시에 방출하므로, 모든 방출선의 선량 기여를 합산한 복합(composite) 반가층을 산정해야 한다.

6.4.1 산정 방법론

Step 1: 방출선별 공기커마 선량률정수 기여분

Γi ∝ Yi × Ei × (μen/ρ)air,i

Yi: 방출률, Ei: 에너지, (μen/ρ)air,i: NIST 공기 질량에너지흡수계수
Step 2: 가중치 (선량 기여 분율)

wi = Γi / Σ Γi

모든 방출선의 Γi 합계 대비 개별 기여 비율
Step 3: 복합 투과율 함수

T(x) = Σ wi × exp(−μi × x)

μi = (μ/ρ)i × ρ : i번째 에너지에서의 선형감쇠계수
Step 4: HVL 수치 풀이 (이분법)

T(HVL) = 0.5 를 만족하는 x = HVL
T(QVL) = 0.25, T(TVL) = 0.1, ...

수치 풀이 방법
다중 지수 합 T(x) = Σ wi × exp(−μix)에 대해 T(x) = 목표값(0.5, 0.25, ...)을 만족하는 x를 이분법(bisection method)으로 구하였다. 허용오차 10−10, 최대 반복 100회.
▶ Python 코드 핵심 로직
# 1. 방출선별 공기커마 기여 가중치 계산
for each emission line (E_i, Y_i):
  muen_air = log_log_interpolate(E_i, NIST_air_table)
  gamma_i = Y_i × E_i × muen_air
w_i = gamma_i / sum(all gamma_i)

# 2. 차폐 투과율 함수
def transmission(x, weights, material):
  T = 0
  for (E_i, w_i) in weights:
    mu_rho = log_log_interpolate(E_i, NIST_material_table)
    mu = mu_rho × rho  # 선형감쇠계수
    T += w_i × exp(−mu × x)
  return T

# 3. 이분법으로 HVL 탐색
x_lo, x_hi = 0, 10  # cm
for 100 iterations:
  x_mid = (x_lo + x_hi) / 2
  if transmission(x_mid) > 0.5:
    x_lo = x_mid
  else:
    x_hi = x_mid
HVL = x_mid

Gemini 컷오프 방식 수용 & 보완

§6.4의 복합 스펙트럼 HVL은 다중 지수 함수의 수치해석(Python bisection)을 필요로 하므로, 규제기관이 수기 검산 가능한 형태를 요구할 경우 적용이 어렵다. Gemini의 교차검증에서 제안된 단일 지배에너지 기반 컷오프 방식은 이 문제를 해결하며, 아래와 같이 보완하여 수용한다.

8.1 컷오프 방식의 원리

핵심 아이디어
각 핵종의 다중 방출선 중 차폐재 투과력이 가장 큰(= 에너지가 높은) 단일 에너지선만으로 HVL을 계산한다.

• 저에너지 X선 성분 제거 → HVL이 커짐 → 차폐 두께가 더 두껍게 산정됨 → 보수적(안전 방향)
• 단일 지수함수 HVL = ln(2)/μ → 계산기만으로 수계산 가능

8.5 컷오프 방식의 적용 한계와 보완

컷오프 방식은 차폐 설계에는 보수적이나, 선량률 평가에는 비보수적이다.

• 차폐(HVL): 저에너지 제거 → HVL 증가 → 더 두꺼운 차폐 요구 → 보수적 ✓
• 선량률(Γ): 저에너지 제거 → Γ 감소 → 선량률 과소평가 → 비보수적 ✗

따라서 선량률 평가에는 S&S 또는 전 스펙트럼 값을 채택하고, 컷오프 HVL은 차폐 검증용으로만 사용해야 한다.

def log_interp(x, x1, y1, x2, y2):
    t = (math.log(x) - math.log(x1)) / (math.log(x2) - math.log(x1))
    return math.exp(math.log(y1) + t * (math.log(y2) - math.log(y1)))

def calc_weights(lines):
    """Calculate dose-rate weight for each emission line"""
    contribs = []
    for E, Y, name in lines:
        muen = get_air_muen(E)
        gi = Y * E * muen  # proportional to Gamma_i
        contribs.append((E, Y, name, muen, gi))
    total = sum(c[4] for c in contribs)
    return [(E, Y, name, muen, gi, gi/total)
            for E, Y, name, muen, gi in contribs], total

def transmission(x_cm, weights, mu_func, rho):
    """Composite transmission: T(x) = Sum[ w_i * exp(-mu_i * x) ]"""
    T = 0
    for E, Y, name, muen, gi, wi in weights:
        mu = mu_func(E) * rho  # linear attenuation coeff [cm^-1]
        T += wi * math.exp(-mu * x_cm)
    return T

def find_thickness(weights, mu_func, rho, target_T):
    """Bisection method to find x where T(x) = target_T"""
    x_lo, x_hi = 0, 10  # cm
    while transmission(x_hi, weights, mu_func, rho) > target_T:
        x_hi *= 2
    for _ in range(100):
        x_mid = (x_lo + x_hi) / 2
        T = transmission(x_mid, weights, mu_func, rho)
        if abs(T - target_T) < 1e-10:
            break
        if T > target_T:
            x_lo = x_mid
        else:
            x_hi = x_mid
    return x_mid
