1. 평가 목적

금번 변경허가 신청에 따라 추가·변경되는 3핵종이 기존 저장시설 경계면 선량률 평가 결과에 미치는 영향을 정량적으로 평가한다.

3. 평가 방법

3.1 기본 가정

비차폐 점선원(bare point source) 가정
선원 인증서가 미확보되어 밀봉캡슐의 재질·두께·형상을 특정할 수 없으므로, 밀봉캡슐에 의한 자기차폐를 전혀 고려하지 않은 최보수적 조건으로 평가한다.
참고: Fe-55 밀봉선원은 5.9 keV Mn Kα X선을 분석용으로 방출하기 위해 Be창 등 저Z 창재를 사용하므로, 캡슐 차폐를 가정하는 것은 부적절하다. 따라서 3핵종 모두 비차폐 선량률 평가로 갈음한다.

3.2 선량률 산정식

Dose Rate = Γ × A / r²
기호의미단위
Γ선량률정수 (공기커마 기준)μSv·m²/(MBq·h)
A방사능량MBq
r평가 거리m

3.3 단위 환산 방법

Smith & Stabin [R1]의 선량률정수는 Exposure Rate Constant (R·cm²/(mCi·h)) 단위로 수록되어 있다. 본 평가에서는 다음 환산 과정을 거쳐 μSv·m²/(MBq·h)로 변환하였다.
환산 체인:

Γ [μSv·m²/(MBq·h)] = ΓR [R·cm²/(mCi·h)] × 0.023703

유도:
  • R → μSv: 1 R = 8.77 mGy = 8,770 μGy ≈ 8,770 μSv (Q=1, 광자)
  • cm² → m²: ÷ 10,000
  • mCi → MBq: ÷ 37 (1 mCi = 37 MBq)

  = 8,770 / 10,000 / 37 = 0.023703
보수성 참고: 이 환산은 공기커마(Ka) 기준이며, 실제 인체 유효선량(effective dose)이나 주위선량당량 H*(10)은 에너지에 따라 이보다 낮거나 비슷한 수준이다. 따라서 공기커마 기준 평가는 보수적이다.

3.4 평가 거리

저장구역 경계면(저장함 표면 ~ 최소 0.3 m) 및 1 m(기준 거리)에서 평가한다.

  (d) 계산 과정

단위 환산: keV → J: ×1.602×10−16 | cm²/g → m²/kg: ×0.1

Step 1: 각 방출선의 기여분

Mn Kα: 0.166 × (5.899 × 1.602×10−16) × (23.89 × 0.1)
  = 0.166 × 9.450×10−16 × 2.389 = 3.749 × 10−16

Mn Kβ: 0.034 × (6.490 × 1.602×10−16) × (17.86 × 0.1)
  = 0.034 × 1.040×10−15 × 1.786 = 6.316 × 10−17
Step 2: 합산 및 4π 나눗셈

합계 = 3.749×10−16 + 6.316×10−17 = 4.381×10−16
ΓSI = 4.381×10−16 / 4π = 4.381×10−16 / 12.566 = 3.486×10−17 (Gy·m²/(Bq·s))
Step 3: 단위환산 → μSv·m²/(MBq·h)

Gy→Sv: ×1 (광자선, Q=1)
Sv→μSv: ×106
Bq→MBq: ×106
s→h: ×3,600
종합 환산계수 = 106 × 106 × 3,600 = 3.6 × 1015
Step 4: 최종값

ΓFe-55 = 3.486 × 10−17 × 3.6 × 1015 = 0.1255 μSv·m²/(MBq·h)

역산 검증: 0.1255 / 3.6×1015 = 3.486×10−17 → ×12.566 = 4.381×10−16 ✓
채택값: ΓFe-55 = 0.1255 μSv·m²/(MBq·h) (직접 산정, [R3]+[R4])
참고 — Fe-55 5.9 keV X선의 실제 의미
• 공기 중 MFP(평균자유행정) ≈ 0.3 m → 1 m 거리에서 대부분 흡수
• 인체 투과 불가 (피부 1 mm 이내 흡수) → 유효선량 기여 사실상 0
• Peplow [R2]의 유효선량 기반 Γ = 2.34×10−14 mSv/(h·MBq) ≈ 0
• 본 평가의 0.1255는 공기커마 기준이며, 실제 피폭 위험은 사실상 없으나 최보수적 값으로 사용

5.4 저장시설 경계면 선량률 변동

기존 저장시설 표면선량률 평가 결과(차폐 포함)에 대해, 추가 3핵종의 비차폐(최보수적) 기여분을 더하더라도 기존 평가 결과의 유효숫자 범위 내이며, 저장시설 표면선량률 평가 결과에 변동 없음.

μ/ρ = e−0.7198 = 0.487 cm²/g
μ = 0.487 × 7.874 = 3.83 cm−1
HVL = 0.693 / 3.83 = 0.181 cm = 1.81 mm

6.4 핵종별 복합 스펙트럼 반가층 산정

§6.2~6.3의 단색 HVL은 개별 방출선 에너지에 대한 것이다. 실제 핵종은 다수의 에너지를 동시에 방출하므로, 모든 방출선의 선량 기여를 합산한 복합(composite) 반가층을 산정해야 한다.

6.4.1 산정 방법론

Step 1: 방출선별 공기커마 선량률정수 기여분

Γi ∝ Yi × Ei × (μen/ρ)air,i

Yi: 방출률, Ei: 에너지, (μen/ρ)air,i: NIST 공기 질량에너지흡수계수
Step 2: 가중치 (선량 기여 분율)

wi = Γi / Σ Γi

모든 방출선의 Γi 합계 대비 개별 기여 비율
Step 3: 복합 투과율 함수

T(x) = Σ wi × exp(−μi × x)

μi = (μ/ρ)i × ρ : i번째 에너지에서의 선형감쇠계수
Step 4: HVL 수치 풀이 (이분법)

T(HVL) = 0.5 를 만족하는 x = HVL
T(QVL) = 0.25, T(TVL) = 0.1, ...

다중 지수 합이므로 해석적 풀이 불가 → Python bisection으로 수치 풀이
비지수적 감쇠 특성
다중 에너지 스펙트럼의 투과율 T(x)는 단일 지수함수가 아니다. 저에너지 X선이 먼저 차폐되면서 얇은 두께에서는 HVL이 작고(빠른 감쇠), 두꺼운 두께에서는 고에너지 γ선만 남아 감쇠가 느려진다. 이 때문에 QVL ≠ 2×HVL, TVL ≠ HVL×ln(10)/ln(2)이다.

6.4.4 Python 계산 코드 및 검증

수치 풀이 방법
다중 지수 합 T(x) = Σ wi × exp(−μix)에 대해 T(x) = 목표값(0.5, 0.25, ...)을 만족하는 x를 이분법(bisection method)으로 구하였다. 허용오차 10−10, 최대 반복 100회.
▶ Python 코드 핵심 로직
# 1. 방출선별 공기커마 기여 가중치 계산
for each emission line (E_i, Y_i):
  muen_air = log_log_interpolate(E_i, NIST_air_table)
  gamma_i = Y_i × E_i × muen_air
w_i = gamma_i / sum(all gamma_i)

# 2. 차폐 투과율 함수
def transmission(x, weights, material):
  T = 0
  for (E_i, w_i) in weights:
    mu_rho = log_log_interpolate(E_i, NIST_material_table)
    mu = mu_rho × rho  # 선형감쇠계수
    T += w_i × exp(−mu × x)
  return T

# 3. 이분법으로 HVL 탐색
x_lo, x_hi = 0, 10  # cm
for 100 iterations:
  x_mid = (x_lo + x_hi) / 2
  if transmission(x_mid) > 0.5:
    x_lo = x_mid
  else:
    x_hi = x_mid
HVL = x_mid

# 입력 데이터
• 방출선: ICRP 107 [R3] — Am-241 10개선, Cd-109 5개선
• 공기 μen/ρ: NIST Air
• 철 μ/ρ: NIST Fe(Z=26)
• 납 μ/ρ: NIST Pb(Z=82)
• ρFe = 7.874 g/cm³, ρPb = 11.35 g/cm³
• 흡수단 처리: Pb L-edges(13.0~15.9 keV), K-edge(88.005 keV) 구간분리 보간

7. 취급 시 접촉(r → 0) 선량률에 관한 고찰

7.1 문제

점선원 모델에서 선원을 손으로 쥘 경우 r → 0이 되어 선량률이 수학적으로 무한대로 발산한다.

7.2 이 문제가 별도 평가를 요하지 않는 근거

IAEA Safety Guide RS-G-1.9 [R7]의 D-value 산정 과정에서는 다음 5가지 피폭 시나리오를 이미 평가하고 있다:

시나리오설명r 조건
1비차폐 선원을 신체에 밀착 휴대r ≈ 수 mm
2비차폐 선원을 손으로 직접 취급r ≈ 수 mm
3비차폐 선원 인근 장기 체류r ~ 1 m
4선원 파손 후 방사성물질 흡입/섭취내부피폭
5선원 분산에 의한 수원 오염내부피폭
즉, D-value에 기반한 등급분류(Am-241 4등급, Cd-109·Fe-55 5등급) 자체가 접촉 포함 최악 시나리오에서의 건강영향을 이미 반영한 결과이다.

등급비차폐 접촉 시 건강영향

5등급 (Cd-109, Fe-55)영구적 손상 가능성 낮음

7.4 기존 취급 현황과의 비교

핵종등급허가량접촉 시 위험도
Co-601등급55.5 TBq수 분 이내 치명적
Ir-1922등급 이상3,788.8 TBq수 시간 이내 치명적
금번 추가되는 4등급(Am-241) 및 5등급(Cd-109, Fe-55) 선원의 접촉 위험은 기존 1·2등급 선원 대비 현저히 낮으므로, 기존 방사선방호 절차로 안전성이 충분히 담보된다.

8. 결론

  저장시설 경계면 선량률

3핵종 추가분은 비차폐(최보수적) 조건에서도 기존 저장선원의 3.4 × 10−6 수준이며, 기존 평가 결과의 유효숫자 범위 내이므로 평가 결과 변동 없음.

  차폐 특성

밀봉캡슐 벽(0.5~2 mm 금속)에 의해 저에너지 X선 성분(Am-241 L X-ray, Cd-109 Ag K X-ray, Fe-55 Mn K X-ray)은 사실상 완전 차폐되며, 실제 선량은 본 비차폐 평가의 극히 일부에 불과하다.

  취급 시 접촉 선량률

IAEA RS-G-1.9 등급분류 체계에 의해 이미 평가되어 있으며, 기존 1·2등급 선원 취급절차로 안전성 담보됨.
